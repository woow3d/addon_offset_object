## ðŸ”¹ Full Description / What it does

**Offset Objects By Name (Axis + MM)** allows you to:

- Offset multiple objects sharing the same base name (Active Object) **or by selection order**.
- Choose axis: X, Y, Z.
- Set distance between objects in millimeters.
- Choose direction: + or -.
- Option to offset the first object or leave it in place.
- Use a multiplier to amplify offset for faster adjustments.
- All settings appear in **Redo Panel / F9**.
